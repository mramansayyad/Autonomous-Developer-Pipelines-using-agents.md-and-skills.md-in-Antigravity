# 🚀 Autonomous Developer Pipelines using Agents & Skills (Google Antigravity)

> Build a fully autonomous AI developer team using **agents.md** and **skills.md** powered by **Google Antigravity IDE**

---

## 📌 Overview

This project demonstrates how to create an **AI-powered autonomous software development pipeline** using the **Google Antigravity framework**.

Instead of writing complex orchestration code, this system uses:
- 🧠 **Agents (`agents.md`)** to define AI roles
- ⚙️ **Skills (`skills.md`)** to define capabilities
- 🔄 **Workflows** to automate execution

The result:  
👉 A complete AI developer team that can take a project from idea → development → deployment automatically.

---

## 🧠 Key Concepts

### 👥 Agents (agents.md)
Defines the AI team roles:
- Product Manager
- Frontend Developer
- Backend Developer
- QA Engineer
- DevOps Engineer

---

### ⚙️ Skills (skills/)
Reusable capabilities defined as modular `.md` files:
- Code generation
- Testing
- Debugging
- Deployment

---

### 🔄 Workflows
Custom command pipelines that:
- Chain multiple agents together
- Automate the entire development lifecycle
- Execute tasks in sequence

## 🧪 How It Works

1. Define your AI team in `agents.md`
2. Add capabilities in `skills/`
3. Create workflows to automate tasks
4. Run the pipeline using Antigravity commands
